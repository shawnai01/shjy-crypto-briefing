## 4. Base B20, stablecoin/RWA용 native token standard를 mainnet에 도입

Base는 Beryl upgrade를 통해 B20 token standard를 도입했습니다. B20은 ERC-20 superset이지만 smart contract가 아니라 Base node software 안의 precompile로 실행되며, role-based access control, supply cap, granular pause, allowlist/blocklist, memo, permit, freeze-and-seize 기능을 기본 제공합니다.

왜 중요한가: Coinbase/Base가 stablecoin, RWA, tokenized equity 발행자를 위해 chain-level compliance primitive를 넣은 것입니다. regulated issuer에는 비용·감사 부담을 줄이지만, 동시에 freeze/admin 권한이 token layer에 내장되므로 투명한 권한 표기와 explorer/indexer 지원이 중요해집니다.

CT/교차검증 코멘트: Base 공식 blog와 docs는 B20이 Base mainnet용 native precompile이고 Base/Spearbit audit을 거쳤다고 설명합니다. Unchained는 block explorer/indexer가 아직 admin 권한을 충분히 읽지 못하는 점을 리스크로 지적했습니다.
